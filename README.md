## Research

**Multi-Task Transfer Learning for Malaria Detection** — *MSc Thesis, 2025*

Built a unified model that handles detection, segmentation, localization, and classification of malaria parasites from thin blood smear images. The multi-task approach showed real gains in robustness compared to running each task separately. Tools: PyTorch, NumPy, OpenCV, Albumentations.
